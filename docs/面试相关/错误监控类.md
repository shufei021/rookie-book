## 错误监控类

#### 一、错误的捕获方式

#### 即时运行错误的捕获方式

+ try..catch (需要写在代码中)
+ window.onerror

#### 资源加载错误 捕获的方式

+ object.onerror 例如 img.onerror
+ performance.getEntries()  获取加载的资源 再通过已经加载的资源 做对比 可以检测出那些资源加载失败
+ Error 捕获事件 

```javascript
window.addEventListener('error',function(err){
    console.log(err)
},true)//一定是true 可以捕获到
```
